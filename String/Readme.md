
Welcome to the String section of my DSA workspace!
This directory contains classic and challenging string problems, many inspired by or directly related to popular LeetCode problems. Each folder may have solutions in C++, Java, and Python.

### 📂 Folders & Problems

- **Isomorphic String**  
	Check if two strings are isomorphic.  
	LeetCode: [205. Isomorphic Strings](https://leetcode.com/problems/isomorphic-strings/)

- **Largest Common Prefix**  
	Find the longest common prefix among strings.  
	LeetCode: [14. Longest Common Prefix](https://leetcode.com/problems/longest-common-prefix/)

- **Largest Odd Number**  
	Find the largest odd number from a string.  
	LeetCode: [1903. Largest Odd Number in String](https://leetcode.com/problems/largest-odd-number-in-string/)

- **Reverse The String**  
	Reverse the given string.  
	LeetCode: [344. Reverse String](https://leetcode.com/problems/reverse-string/)

- **Integer To Roman**  
	Convert integer to Roman numeral.  
	LeetCode: [12. Integer to Roman](https://leetcode.com/problems/integer-to-roman/)

- **Max Nesting Depth Parenthesis**  
	Find the maximum nesting depth of parentheses.  
	LeetCode: [1614. Maximum Nesting Depth of the Parentheses](https://leetcode.com/problems/maximum-nesting-depth-of-the-parentheses/)

- **Remove Outermost Parenthesis**  
	Remove the outermost parentheses.  
	LeetCode: [1021. Remove Outermost Parentheses](https://leetcode.com/problems/remove-outermost-parentheses/)

- **Roman To Integer**  
	Convert Roman numeral to integer.  
	LeetCode: [13. Roman to Integer](https://leetcode.com/problems/roman-to-integer/)

- **String To Integer**  
	Convert string to integer (atoi).  
	LeetCode: [8. String to Integer (atoi)](https://leetcode.com/problems/string-to-integer-atoi/)

- **Substring With K Distinct Characters**  
	Count substrings with exactly K distinct characters.

- **Valid Anagram**  
	Check if two strings are anagrams.  
	LeetCode: [242. Valid Anagram](https://leetcode.com/problems/valid-anagram/)

- **KMP Algorithm**  
	Knuth-Morris-Pratt substring search.  
	LeetCode: [28. Find the Index of the First Occurrence in a String](https://leetcode.com/problems/find-the-index-of-the-first-occurrence-in-a-string/)

- **Longest Substring Without Repeating Characters**  
	Find the length of the longest substring without repeating characters.  
	LeetCode: [3. Longest Substring Without Repeating Characters](https://leetcode.com/problems/longest-substring-without-repeating-characters/)
